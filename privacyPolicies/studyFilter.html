Privacy Policy

Last updated: July 2025

This privacy policy explains how the StudyFilter Chrome extension collects and uses data.

Data Collection

StudyFilter does not collect personally identifiable information from users.

It processes only the following data locally or via secure API calls:

  • Website URLs and titles for filtering purposes.
  • Search phrases and user-defined topics to provide filtering recommendations.

Data Usage

Data processed is used solely to provide content filtering features and improve user experience. No data is shared with third parties.

Data Storage

No personal data is stored by StudyFilter beyond temporary processing for filtering logic.

Third-Party Services

StudyFilter uses OpenAI APIs to evaluate relevance of content. Requests to OpenAI are limited to non-identifiable phrases and topics only.

Your Choices

You may control filtering settings within the extension and can disable or uninstall StudyFilter at any time.

Contact

If you have questions about this privacy policy, please contact Spencer Ng via spenzcfc@gmail.com.

This policy applies only to the StudyFilter Chrome extension.
